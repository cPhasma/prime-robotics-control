# krpff_checks

Вспомогательные утилиты для проверки аппаратной части проекта.

## Состав

```text
pwm_mapping_check.py      проверка инверсного PWM
sensor_log_analyzer.py   анализ CSV-логов датчиков
run_checks.py            общий запуск проверок
sample_sensor_log.csv    пример входного лога
```

## Назначение

Утилиты используются для быстрой проверки отдельных проблем, которые возникали при работе системы:

- инверсная логика PWM;
- временные пропуски данных дальномера;
- проверка периода опроса датчиков;
- анализ корректности данных перед запуском регулятора.

## Формат CSV-лога

Минимальные поля:

```text
time_ms,distance_m,gyro_z
```

Пример:

```text
0,1.20,0.001
50,1.18,0.002
100,,0.001
150,1.15,
```

Пустое значение считается пропуском измерения.
