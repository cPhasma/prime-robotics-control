# v01_two_cars_base — Базовая версия управления двумя машинками

## Кратко

Регулятор вынесен из server.py в controllers, добавлена поддержка car1/car2, параллельный запуск, защита от кратковременной ошибки дальномера. Историческая версия: PWM описан как обычный Arduino PWM, потом исправлялся.

## Исходный архив

`site_two_cars_fixed (1).zip`

## Как использовать

Эта папка является историческим снимком версии. Для запуска итоговой версии лучше использовать корень репозитория.
