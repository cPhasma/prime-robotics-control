Этого кода не было в архиве, но для надёжной работы скриптов обязательно доправить прошивки.

1) ESP8266 bridge:
- Убрать ВСЕ Serial.print/println отладочного текста, если этот UART идёт в Arduino.
- Заменить readStringUntil('\n') на накопительный буфер по символам.
- Пересылать только чистые JSON-строки сервер <-> Arduino.

2) Arduino UNO:
- Экранировать stdout/stderr в command_result JSON.
- Для motor сделать короткий stdout вроде "ok".
- Лучше заменить блокирующее Serial.readStringUntil('\n') на накопительный буфер.
- Команда stop должна реально отправлять pwm=0, а не 255.

3) Архитектура:
- Одна команда в полёте за раз.
- Скрипт отправляет следующую команду только после ACK предыдущей по command_id.
