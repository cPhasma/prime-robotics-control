# v07_experimental_sensor_filter — Экспериментальная правка фильтрации датчиков

## Кратко

Экспериментальная ветка по задержке контура, фильтрации MPU6050 и переезду целевого расстояния. Сохранена как отдельный snapshot, потому что структура отличается: регулятор частично находится в server.py.

## Исходный архив

`L2_follow_controller_sensor_filter_fix (1).zip`

## Как использовать

Эта папка является историческим снимком версии. Для запуска итоговой версии лучше использовать корень репозитория.
