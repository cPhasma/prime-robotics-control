# Patch: реальный регулятор следования

Добавлен серверный регулятор следования за объектом прямо в веб-интерфейс.

## Где находится

- `server.py` — добавлен `FollowController` и WebSocket-события:
  - `follow_start`
  - `follow_stop`
  - `follow_update_params`
  - `follow_telemetry`
  - `follow_state`
  - `follow_error`
- `templates/index.html` — добавлена панель «Реальное следование за объектом» с параметрами, стартом/стопом и графиками.

## Вектор состояния

Регулятор в реальном времени ведёт оценку состояния:

```text
x = [vL_hat, vR_hat, theta, z, d]
```

- `d` берётся с HC-SR04 через команду `duration`.
- `gyro_z` берётся с MPU6050 через команду `mpu6050`.
- `theta` считается интегрированием `gyro_z` после калибровки bias.
- `z` считается по кинематике `z += v * sin(theta) * dt`.
- `vL_hat`, `vR_hat` считаются по модели двигателя первого порядка.

## Закон управления

Продольный канал:

```text
u_v = vf_hat / K + kp_d * (d - d_target) + kd_d * d_dot
```

где:

```text
vf_hat = d_dot + v
```

Курсовой канал:

```text
u_w = -k_z * z - k_theta * theta - k_omega * gyro_z
```

Управление бортами:

```text
uL = u_v - u_w
uR = u_v + u_w
```

## Важное про PWM

В текущей прошивке проекта `cmd_motor.cpp` использует `analogWrite(pwm)`, а в старом guide указано, что stop = `PWM=0`. Поэтому в UI по умолчанию стоит режим:

```text
0 = stop, 255 = max
```

Если на реальной машинке управление действительно инвертировано:

```text
0 = max, 255 = stop
```

включи в панели параметр `PWM инвертирован = да` перед стартом. После этого аварийный stop в server.py тоже будет отправлять `PWM=255`.

## Как тестировать

1. Запустить сервер:

```bash
python server.py
```

2. Открыть:

```text
http://localhost:5000
```

3. Дождаться `ESP: Connected`.
4. Поставить машинку неподвижно на 1–2 секунды.
5. В панели «Реальное следование за объектом» выставить целевую дистанцию, например `1.0` м.
6. Проверить режим PWM.
7. Нажать `Старт регулятора`.
8. Смотреть графики `d`, `error`, `theta`, `z`, `u`, `PWM`.
9. Для остановки использовать кнопку `Стоп`.
