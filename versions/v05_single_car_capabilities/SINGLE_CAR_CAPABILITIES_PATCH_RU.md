# Патч: работа с одной машинкой и список команд

Что исправлено:

1. Сервер больше не считает ошибкой отсутствие второй машинки, когда запрашивается список команд для `all`.
   Теперь запрос отправляется только подключённым машинкам.

2. Если подключена только одна машинка, сайт может сразу работать с ней: `car1` или `car2`.

3. После подключения ESP сервер автоматически просит Arduino повторно отправить список команд:

```json
{"cmd":"get_capabilities","command_id":"..."}
```

4. ESP-прошивки `car1` и `car2` тоже после подключения к серверу сами отправляют Arduino запрос `get_capabilities`.
   Это нужно, потому что Arduino часто отправляет capabilities при старте раньше, чем ESP успевает подключиться к серверу.

5. Добавлен резервный список команд на стороне сервера. Он временно показывается на сайте, если реальный список от Arduino ещё не пришёл. Когда Arduino отвечает, список заменяется реальным.

6. Кнопка запуска двух регуляторов теперь запускает регуляторы только на подключённых машинках. Если подключена одна машинка, запускается только она, без ошибки про вторую.
