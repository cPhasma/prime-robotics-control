# v06_matrix_model_full — Итоговая матричная модель и общий опрос датчиков

## Кратко

Итоговая полная версия: x(k+1)=Ad*x+Bd*u+Gd*vf+Hd*omega_gyro, y=C*x, q=F*xi, u=S*q. Добавлена команда sensors для совместного опроса HC-SR04 и MPU6050.

## Исходный архив

`L2_follow_controller_matrix_model_full (1).zip`

## Как использовать

Эта папка является историческим снимком версии. Для запуска итоговой версии лучше использовать корень репозитория.
