# Patch: регулятор следования и две машинки

Актуальная версия вынесла регуляторы из `server.py` и добавила параллельную работу двух машинок.

## Где находится код

- `server.py` — сайт, TCP-подключения, маршрутизация команд на `car1` / `car2`.
- `controllers/follow_base.py` — общая база регулятора: поток, обмен, фильтры, защита от ошибок датчиков, графики и логи.
- `controllers/follow_car1.py` — алгоритм регулятора первой машинки.
- `controllers/follow_car2.py` — алгоритм регулятора второй машинки.
- `templates/index.html` — сайт с отдельными графиками для `car1` и `car2`.

## PWM

Используется инверсная логика вашей прошивки/драйвера:

```text
0 = максимум
255 = стоп / минимальная скорость
```

Инверсная логика включена в серверном регуляторе: управляющее `u=0` превращается в `PWM=255`, а `u=1` превращается в `PWM=0`.

## Защита от ошибок дальномера

Если HC-SR04 на мгновение потерял эхо, Arduino возвращает `valid=false`, а серверный регулятор не падает. Он берёт последнее корректное расстояние, пишет warning в лог и продолжает работу.

## Параллельный старт

На сайте есть кнопка:

```text
▶▶ Старт двух регуляторов
```

Она запускает два отдельных потока: один для `car1`, второй для `car2`.
