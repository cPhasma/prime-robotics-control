# v04_mpu6050_i2cdev — MPU6050 через I2Cdev

## Кратко

MPU6050 переведён с Adafruit_MPU6050 на I2Cdevlib, добавлена однократная калибровка CalibrateGyro(6), команда возвращает угловую скорость в rad/s.

## Исходный архив

`site_two_cars_mpu6050_i2cdev_fixed.zip`

## Как использовать

Эта папка является историческим снимком версии. Для запуска итоговой версии лучше использовать корень репозитория.
