# Участники проекта

## SouthBoard

Основная работа по проекту:

- сборка и структурирование репозитория;
- работа с серверной частью управления;
- оформление версий проекта;
- описание матричной модели объекта;
- разбор логики регулятора следования;
- подготовка технической документации;
- интеграция кода сайта, Arduino/ESP и модулей регулятора.

## @krpff

Дополнительная техническая работа:

- проверка поведения системы на тестовых сценариях;
- анализ логики инверсного PWM-управления;
- проверка реакции алгоритма на временные ошибки дальномера;
- участие в разборе работы датчиков HC-SR04 и MPU6050;
- подготовка вспомогательных утилит для проверки PWM и логов датчиков;
- фиксация замечаний по работе одной и двух машинок.

Основная зона ответственности — тестовые проверки, анализ поведения системы на железе и вспомогательные инструменты для контроля корректности данных.
