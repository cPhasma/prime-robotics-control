# Проектное досье

## Тема

Программное управление мобильными роботами команды «Прайм».

## Цель

Собрать систему, которая позволяет управлять одной или двумя машинками через сайт, получать данные датчиков и запускать алгоритмы следования.

## Основные компоненты

- Flask-сервер;
- веб-интерфейс;
- ESP/Arduino;
- моторы с PWM-управлением;
- дальномер HC-SR04;
- гироскоп MPU6050;
- регуляторы следования.

## Выполненные этапы

1. Подготовлена структура сайта управления.
2. Добавлена работа с двумя машинками.
3. Регуляторы вынесены в отдельные файлы.
4. Исправлена инверсная логика PWM.
5. Добавлена обработка временных ошибок дальномера.
6. MPU6050 переведён на I2Cdev.
7. Добавлена однократная калибровка гироскопа.
8. Исправлена работа с одной подключённой машинкой.
9. Добавлена команда `sensors`.
10. Описана матричная модель объекта и закон управления.

## Технический результат

Проект приведён к структуре, в которой серверная часть, контроллеры, Arduino-код, архивы версий и документация разделены по папкам. Это упрощает дальнейшую отладку и развитие системы.
