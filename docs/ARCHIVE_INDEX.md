| Архив | Связанная версия | Назначение |
|---|---|---|
| `site_two_cars_fixed (1).zip` | `v01_two_cars_base` | Базовая версия для двух машинок. |
| `site_two_cars_pwm_inverted_fixed.zip` | `v02_pwm_inverted_fix` | Исправление PWM. |
| `site_two_cars_math_control_fixed.zip` | `v03_matrix_control_law` | Матричный закон управления. |
| `site_two_cars_mpu6050_i2cdev_fixed.zip` | `v04_mpu6050_i2cdev` | MPU6050 через I2Cdev. |
| `site_single_car_capabilities_fixed.zip` | `v05_single_car_capabilities` | Поддержка одной машинки и список команд. |
| `L2_follow_controller_matrix_model_full (1).zip` | `v06_matrix_model_full` | Итоговая матричная модель. |
| `L2_follow_controller_sensor_filter_fix (1).zip` | `v07_experimental_sensor_filter` | Экспериментальная фильтрация датчиков. |

Контрольные суммы архивов сохранены в файле `archive/SHA256SUMS.txt`.
