# CHANGELOG

История версий будет добавлена по мере коммитов.

## v01_two_cars_base — Базовая версия управления двумя машинками

Регулятор вынесен из server.py в controllers, добавлена поддержка car1/car2, параллельный запуск, защита от кратковременной ошибки дальномера. Историческая версия: PWM описан как обычный Arduino PWM, потом исправлялся.


## v02_pwm_inverted_fix — Исправление инверсной PWM-логики

Сервер приведён к фактической логике прошивки/драйвера: PWM=0 — максимум, PWM=255 — стоп. Исправлены stop_pwm, norm_to_pwm, аварийная остановка и подписи на сайте.


## v03_matrix_control_law — Матричный закон управления

Закон управления в follow_car1.py/follow_car2.py переписан через матричное умножение: q=F*xi, u=S*q. Условия if/elif вынесены из самой функции формирования управления.


## v04_mpu6050_i2cdev — MPU6050 через I2Cdev

MPU6050 переведён с Adafruit_MPU6050 на I2Cdevlib, добавлена однократная калибровка CalibrateGyro(6), команда возвращает угловую скорость в rad/s.


## v05_single_car_capabilities — Работа с одной машинкой и capabilities

Сервер больше не падает, если подключена только одна машинка. Запрос get_capabilities отправляется только реально подключённым машинкам, добавлен резервный список команд.


## v06_matrix_model_full — Итоговая матричная модель и общий опрос датчиков

Итоговая полная версия: x(k+1)=Ad*x+Bd*u+Gd*vf+Hd*omega_gyro, y=C*x, q=F*xi, u=S*q. Добавлена команда sensors для совместного опроса HC-SR04 и MPU6050.
