# Журнал изменений

## v01_two_cars_base

Базовая версия системы управления двумя машинками.

Изменения:

- добавлена структура для работы с `car1` и `car2`;
- регулятор следования вынесен из `server.py` в отдельные файлы;
- добавлена возможность запускать два регулятора параллельно;
- добавлена защита от кратковременной ошибки дальномера;
- сохранены логи работы регулятора в терминале.

## v02_pwm_inverted_fix

Исправлена логика PWM под используемую прошивку моторов.

Изменения:

- учтено, что `PWM = 0` соответствует максимальной скорости;
- учтено, что `PWM = 255` соответствует остановке;
- исправлены функции перевода нормированного управления в PWM;
- аварийная остановка переведена на `PWM = 255`;
- слайдеры ручного управления по умолчанию поставлены в безопасное положение.

## v03_matrix_control_law

Закон управления переписан через матричные операции.

Изменения:

- введён расширенный вектор состояния `xi`;
- продольное и угловое управление вычисляются через матрицу `F`;
- управление бортами получается через матрицу `S`;
- из функции формирования управления убраны ветвления, влияющие на сам закон;
- служебная логика запуска, остановки и обработки ошибок оставлена отдельно от математического закона.

## v04_mpu6050_i2cdev

Обновлена работа с гироскопом MPU6050.

Изменения:

- вместо `Adafruit_MPU6050` используется `I2Cdev` и `MPU6050`;
- добавлена однократная калибровка гироскопа при первом обращении;
- данные гироскопа возвращаются в `rad/s`, а не в сырых единицах;
- серверная компенсация смещения отключена, так как калибровка выполняется на Arduino;
- увеличен timeout первого запроса, потому что калибровка занимает время.

## v05_single_car_capabilities

Исправлена работа системы при подключении только одной машинки.

Изменения:

- сервер не требует обязательного подключения второй машинки;
- запуск регулятора выполняется только для реально подключённых машинок;
- запрос списка команд отправляется подключённым ESP;
- добавлен резервный список команд, пока Arduino не ответила;
- ESP при старте запрашивает у Arduino список возможностей.

## v06_matrix_model_full

Итоговая версия с матричной моделью объекта и объединённым опросом датчиков.

Изменения:

- в `follow_base.py` добавлено обновление состояния в матричном виде;
- добавлены матрицы `Ad`, `Bd`, `Gd`, `C`;
- добавлена команда `sensors` для совместного получения расстояния и угловой скорости;
- сохранена инверсная PWM-схема;
- структура регуляторов `car1` и `car2` приведена к единому виду.

## v07_experimental_sensor_filter

Экспериментальная версия по обработке датчиков и уменьшению переезда целевого расстояния.

Изменения:

- рассматривалась проблема бокового отклонения;
- рассматривалась проблема переезда целевой дистанции;
- проверялась идея фильтрации измерений и уменьшения задержки реакции;
- версия сохранена отдельно, так как отличается от основной финальной структуры.
